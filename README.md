# 🛒 Lista de Compras Pro

Aplicativo web para gerenciamento de listas de compras em tempo real, utilizando Firebase Realtime Database.

## Funcionalidades
- **Login de Usuário**: Acesso restrito via credenciais.
- **Gerenciamento de Itens**: Adição, edição e exclusão de produtos com categorias e unidades.
- **Modo de Compra**: Clique em um item para definir o preço pago e marcar como comprado.
- **Histórico**: Armazenamento de listas finalizadas para consulta ou restauração futura.
- **Compartilhamento**: Envio da lista detalhada via WhatsApp.

## Tecnologias
- Tailwind CSS (UI)
- Font Awesome (Ícones)
- Firebase SDK (Banco de dados e Hosting)
